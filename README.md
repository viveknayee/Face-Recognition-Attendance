# 🎓 Face Recognition Attendance System

A smart attendance system that uses face recognition 
to automatically mark attendance. Built with Python and Flask.

## 🛠️ Technologies Used
- Python
- Flask
- OpenCV
- face_recognition library
- SQLite Database

## ⚙️ How to Run

1. Clone this repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open browser: http://localhost:5000
